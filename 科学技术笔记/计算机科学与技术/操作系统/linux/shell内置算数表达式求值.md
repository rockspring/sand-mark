# shell内置算数表达式求值

使用```((表达式))```执行算数表达式

使用```$((表达式))```执行算数表达式

## 1 加法运算

```
(( 1 + 2 ))

result=$(( 1 + 2 )) # result、=、$、((四个字符串之间不能有空格
echo $result
```
