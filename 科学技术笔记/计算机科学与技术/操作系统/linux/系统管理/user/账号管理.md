# 账号管理

## 创建一个group

groupadd work

## 获取group的ID

cat /etc/group

## 创建一个用户

useradd work -g work -d /home/work -m

useradd work -g work -m

-m 创建账号的主目录/home/work

## 删除账号

userdel work

删除账号和组，不会删除主目录


### 通过root账号切换到work账号

su - work
